# Lang Methods

<Paragraph>
    本文档介绍 Lodash 中 <Mark bold>Lang</Mark> 命名空间下的类型检查与类型转换相关方法，涵盖浅拷贝、深拷贝、类型判断、值比较以及基础类型转换等核心能力，帮助开发者快速掌握并正确使用这些方法。
</Paragraph>

<Divider />

## 克隆方法

### _.clone
<Paragraph>
    <Code>_.clone(value)</Code> 用于创建 <Mark bold>value</Mark> 的浅拷贝。该方法基于结构化克隆算法实现，支持数组、数组缓冲、布尔值、日期、Map、数字、普通对象、正则、Set、字符串、Symbol、类型化数组等类型；函数、DOM 节点、错误对象、WeakMap 等不可克隆的值会返回空对象。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要克隆的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回克隆后的值。
</Paragraph>

#### 示例
```js
var objects = [{ 'a': 1 }, { 'b': 2 }];

var shallow = _.clone(objects);
console.log(shallow[0] === objects[0]);
// => true
```

---

### _.cloneDeep
<Paragraph>
    <Code>_.cloneDeep(value)</Code> 与 <Code>_.clone</Code> 类似，但会<Mark bold>递归克隆</Mark>所有嵌套值，生成完全独立的深拷贝对象。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要递归克隆的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回深克隆后的值。
</Paragraph>

#### 示例
```js
var objects = [{ 'a': 1 }, { 'b': 2 }];

var deep = _.cloneDeep(objects);
console.log(deep[0] === objects[0]);
// => false
```

---

### _.cloneWith
<Paragraph>
    <Code>_.cloneWith(value, [customizer])</Code> 是 <Code>_.clone</Code> 的可定制版本，允许传入 <Mark bold>customizer</Mark> 函数自定义克隆逻辑。如果 customizer 返回 undefined，则使用默认克隆逻辑。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要克隆的值</TableCell>
    </TableRow>
    <TableRow>
        <TableCell>[customizer]</TableCell>
        <TableCell>自定义克隆的函数，最多接收四个参数：(value [, index|key, object, stack])</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回克隆后的值。
</Paragraph>

#### 示例
```js
function customizer(value) {
  if (_.isElement(value)) {
    return value.cloneNode(false);
  }
}

var el = _.cloneWith(document.body, customizer);

console.log(el === document.body);
// => false
console.log(el.nodeName);
// => 'BODY'
console.log(el.childNodes.length);
// => 0
```

---

### _.cloneDeepWith
<Paragraph>
    <Code>_.cloneDeepWith(value, [customizer])</Code> 是 <Code>_.cloneWith</Code> 的递归版本，会对所有嵌套值应用自定义克隆逻辑。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要递归克隆的值</TableCell>
    </TableRow>
    <TableRow>
        <TableCell>[customizer]</TableCell>
        <TableCell>自定义克隆的函数</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回深克隆后的值。
</Paragraph>

#### 示例
```js
function customizer(value) {
  if (_.isElement(value)) {
    return value.cloneNode(true);
  }
}

var el = _.cloneDeepWith(document.body, customizer);

console.log(el === document.body);
// => false
console.log(el.nodeName);
// => 'BODY'
console.log(el.childNodes.length);
// => 20
```

## 类型判断方法

### _.isArray
<Paragraph>
    <Code>_.isArray(value)</Code> 用于检查 <Mark bold>value</Mark> 是否为数组类型。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要检查的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    如果 value 是数组则返回 <Mark bold>true</Mark>，否则返回 false。
</Paragraph>

#### 示例
```js
_.isArray([1, 2, 3]);
// => true

_.isArray(document.body.children);
// => false

_.isArray('abc');
// => false

_.isArray(_.noop);
// => false
```

---

### _.isObject
<Paragraph>
    <Code>_.isObject(value)</Code> 用于检查 <Mark bold>value</Mark> 是否为对象类型（包括数组、函数、正则、`new Number(0)`、`new String('')` 等）。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要检查的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    如果 value 是对象则返回 <Mark bold>true</Mark>，否则返回 false。
</Paragraph>

#### 示例
```js
_.isObject({});
// => true

_.isObject([1, 2, 3]);
// => true

_.isObject(_.noop);
// => true

_.isObject(null);
// => false
```

```js
// => true

_.isNumber(Number.MIN_VALUE);
// => true

_.isNumber(Infinity);
// => true

_.isNumber('3');
// => false
```

## 值比较方法

### _.isEqual
<Paragraph>
    <Code>_.isEqual(value, other)</Code> 用于<Mark bold>深度比较两个值</Mark>是否等价，支持数组、数组缓冲、布尔值、日期、错误对象、Map、数字、普通对象、正则、Set、字符串、Symbol、类型化数组等类型，函数和 DOM 节点使用严格相等（===）比较。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要比较的值</TableCell>
    </TableRow>
    <TableRow>
        <TableCell>other</TableCell>
        <TableCell>另一个要比较的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    如果两个值等价则返回 <Mark bold>true</Mark>，否则返回 false。
</Paragraph>

#### 示例
```js
var object = { 'a': 1 };
var other = { 'a': 1 };

_.isEqual(object, other);
// => true

object === other;
// => false
```

---

### _.isEqualWith
<Paragraph>
    <Code>_.isEqualWith(value, other, [customizer])</Code> 是 <Code>_.isEqual</Code> 的可定制版本，允许传入 <Mark bold>customizer</Mark> 自定义比较逻辑。如果 customizer 返回 undefined，则使用默认比较逻辑。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要比较的值</TableCell>
    </TableRow>
    <TableRow>
        <TableCell>other</TableCell>
        <TableCell>另一个要比较的值</TableCell>
    </TableRow>
    <TableRow>
        <TableCell>[customizer]</TableCell>
        <TableCell>自定义比较的函数，最多接收六个参数：(objValue, othValue [, index|key, object, other, stack])</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    如果两个值等价则返回 <Mark bold>true</Mark>，否则返回 false。
</Paragraph>

#### 示例
```js
function isGreeting(value) {
  return /^h(?:i|ello)$/.test(value);
}

function customizer(objValue, othValue) {
  if (isGreeting(objValue) && isGreeting(othValue)) {
    return true;
  }
}

var array = ['hello', 'goodbye'];
var other = ['hi', 'goodbye'];

_.isEqualWith(array, other, customizer);
// => true
```

## 类型转换方法

### _.toArray
<Paragraph>
    <Code>_.toArray(value)</Code> 用于将 <Mark bold>value</Mark> 转换为数组。对象会被转换为自身可枚举字符串键对应的值数组，字符串会被拆分为单个字符数组，其他不可迭代的值返回空数组。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要转换的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回转换后的数组。
</Paragraph>

#### 示例
```js
_.toArray({ 'a': 1, 'b': 2 });
// => [1, 2]

_.toArray('abc');
// => ['a', 'b', 'c']

_.toArray(1);
// => []

_.toArray(null);
// => []
```

---

### _.toNumber
<Paragraph>
    <Code>_.toNumber(value)</Code> 用于将 <Mark bold>value</Mark> 转换为数字类型。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要转换的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回转换后的数字。
</Paragraph>

#### 示例
```js
_.toNumber(3.2);
// => 3.2

_.toNumber(Number.MIN_VALUE);
// => 5e-324

_.toNumber(Infinity);
// => Infinity

_.toNumber('3.2');
// => 3.2
```

### _.toString
<Paragraph>
    <Code>_.toString(value)</Code> 用于将 <Mark bold>value</Mark> 转换为字符串。null 和 undefined 会返回空字符串，-0 的符号会被保留。
</Paragraph>

#### 参数
<Table>
    <TableRow>
        <TableCell>value</TableCell>
        <TableCell>要转换的值</TableCell>
    </TableRow>
</Table>

#### 返回值
<Paragraph>
    返回转换后的字符串。
</Paragraph>

#### 示例
```js
_.toString(null);
// => ''

_.toString(-0);
// => '-0'

_.toString([1, 2, 3]);
// => '1,2,3'
```

## 其他常用 Lang 方法

### _.castArray
<Paragraph>
    <Code>_.castArray(value)</Code> 用于将 <Mark bold>value</Mark> 转换为数组，如果 value 已经是数组则直接返回原数组，否则将其包裹为数组。
</Paragraph>

#### 示例
```js
_.castArray(1);
// => [1]

_.castArray({ 'a': 1 });
// => [{ 'a': 1 }]

_.castArray('abc');
// => ['abc']

_.castArray(null);
// => [null]

var array = [1, 2, 3];
console.log(_.castArray(array) === array);
// => true
```

---

### _.isNil
<Paragraph>
    <Code>_.isNil(value)</Code> 用于检查 <Mark bold>value</Mark> 是否为 null 或 undefined。
</Paragraph>

#### 示例
```js
_.isNil(null);
// => true

_.isNil(void 0);
// => true

_.isNil(NaN);
// => false
```

---

### _.isNull
<Paragraph>
    <Code>_.isNull(value)</Code> 用于检查 <Mark bold>value</Mark> 是否为 null。

```js
_.isNull(null);
```

---

### _.isUndefined
<Paragraph>
    <Code>_.isUndefined(value)</Code> 用于检查 <Mark bold>value</Mark> 是否为 undefined。

#### 示例
```js
_.isUndefined(void 0);
```

---

### _.isEmpty
<Paragraph>
    <Code>_.isEmpty(value)</Code> 用于检查 <Mark bold>value</Mark> 是否为空。对象如果没有自身可枚举字符串键则为空；类数组值（arguments、数组、缓冲、字符串等）长度为 0 则为空；Map 和 Set 大小为 0 则为空。
</Paragraph>

#### 示例
```js
_.isEmpty(null);
// => true

_.isEmpty(true);
// => true

_.isEmpty(1);
// => true

_.isEmpty([1, 2, 3]);
// => false

_.isEmpty({ 'a': 1 });
// => false
```
