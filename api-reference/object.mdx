# Object Methods

Lodash 提供了一系列用于操作 JavaScript 对象的方法，覆盖属性赋值、查询、遍历、过滤、路径操作等核心场景。本文档将详细说明 `assign`、`create`、`defaults`、`toPairs`、`keys`、`values`、`omit`、`pick`、`update`、`set`、`unset` 等常用对象方法的功能、参数、返回值及使用示例。

## assign

<Heading level="2">
    assign
</Heading>

<Paragraph>
    <Mark bold>_.assign(object, [sources])</Mark>
</Paragraph>

<p>将源对象自身可枚举的字符串键属性分配到目标对象。源对象从左到右应用，后续源对象的属性会覆盖前面源对象的同名属性。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：该方法会修改 object，基于 <Link href="https://mdn.io/Object/assign">Object.assign</Link> 实现。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：目标对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            [sources]
        </TableCell>
        <TableCell>
            (...Object)：源对象
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回 object。
</Paragraph>

### 示例
```js
function Foo() {
  this.a = 1;
}

function Bar() {
  this.c = 3;
}

Foo.prototype.b = 2;
Bar.prototype.d = 4;

_.assign({ 'a': 0 }, new Foo, new Bar);
// => { 'a': 1, 'c': 3 }
```

## create

<Heading level="2">
    create
</Heading>

<Paragraph>
    <Mark bold>_.create(prototype, [properties])</Mark>
</Paragraph>

<p>创建一个从 prototype 对象继承的新对象。如果提供了 properties 对象，会将其自身可枚举的字符串键属性分配到新创建的对象上。</p>

### 参数
<Table>
    <TableRow>
        <TableCell>
            prototype
        </TableCell>
        <TableCell>
            (Object)：要继承的原型对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            [properties]
        </TableCell>
        <TableCell>
            (Object)：要分配到新对象的属性
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回新创建的对象。
</Paragraph>

### 示例
```js
function Shape() {
  this.x = 0;
  this.y = 0;
}

function Circle() {
  Shape.call(this);
}

Circle.prototype = _.create(Shape.prototype, {
  'constructor': Circle
});

var circle = new Circle;
circle instanceof Circle;
// => true

circle instanceof Shape;
// => true
```

## defaults

<Heading level="2">
    defaults
</Heading>

<Paragraph>
    <Mark bold>_.defaults(object, [sources])</Mark>
</Paragraph>

<p>将源对象自身和继承的可枚举字符串键属性分配到目标对象，仅当目标对象的对应属性值为 undefined 时才会被赋值。源对象从左到右应用，一旦属性被设置，后续同名属性会被忽略。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：该方法会修改 object。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：目标对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            [sources]
        </TableCell>
        <TableCell>
            (...Object)：源对象
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回 object。
</Paragraph>

### 示例
```js
_.defaults({ 'a': 1 }, { 'b': 2 }, { 'a': 3 });
// => { 'a': 1, 'b': 2 }
```

## toPairs

<Heading level="2">
    toPairs
</Heading>

<Paragraph>
    <Mark bold>_.toPairs(object)</Mark>
</Paragraph>

<p>创建一个包含 object 自身可枚举字符串键值对数组，可用于被 <Link href="#frompairs">_.fromPairs</Link> 消费。如果 object 是 Map 或 Set，会返回其条目。</p>

<Paragraph>
    别名：<Mark italic>_.entries</Mark>
</Paragraph>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：要查询的对象
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回键值对数组。
</Paragraph>

### 示例
```js
function Foo() {
  this.a = 1;
  this.b = 2;
}

Foo.prototype.c = 3;

_.toPairs(new Foo);
// => [['a', 1], ['b', 2]] (迭代顺序不保证)
```

## keys

<Heading level="2">
    keys
</Heading>

<Paragraph>
    <Mark bold>_.keys(object)</Mark>
</Paragraph>

<p>创建一个包含 object 自身可枚举属性名的数组。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：非对象值会被强制转换为对象。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：要查询的对象
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回属性名数组。
</Paragraph>

### 示例
```js
function Foo() {
  this.a = 1;
  this.b = 2;
}

Foo.prototype.c = 3;

_.keys(new Foo);
// => ['a', 'b'] (迭代顺序不保证)

_.keys('hi');
// => ['0', '1']
```

</Paragraph>

<p>创建一个包含 object 自身可枚举字符串键属性值的数组。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：非对象值会被强制转换为对象。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>

        <TableCell>

        </TableCell>

<Paragraph>

</Paragraph>

```js
function Foo() {
```

## omit

<Heading level="2">
    omit
</Heading>

<Paragraph>
    <Mark bold>_.omit(object, [paths])</Mark>
</Paragraph>

<p>与 <Link href="#pick">_.pick</Link> 相反，该方法创建一个新对象，排除 object 中指定的自身和继承的可枚举字符串键路径。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：该方法比 <Link href="#pick">_.pick</Link> 性能差很多。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：源对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            [paths]
        </TableCell>
        <TableCell>
            (...(string|string[]))：要排除的属性路径
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回新对象。
</Paragraph>

### 示例
```js
var object = { 'a': 1, 'b': '2', 'c': 3 };

_.omit(object, ['a', 'c']);
// => { 'b': '2' }
```

## update

<Heading level="2">
    update
</Heading>

<Paragraph>
    <Mark bold>_.update(object, path, updater)</Mark>
</Paragraph>

<p>该方法类似 <Link href="#set">_.set</Link>，但接受 updater 函数来生成要设置的值。可通过 <Link href="#updatewith">_.updateWith</Link> 自定义路径创建。updater 调用时传入一个参数：(value)。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：该方法会修改 object。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：要修改的对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            path
        </TableCell>
        <TableCell>
            (Array|string)：要更新的属性路径
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            updater
        </TableCell>
        <TableCell>
            (Function)：用于生成更新值的函数
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回 object。
</Paragraph>

### 示例
```js
var object = { 'a': [{ 'b': { 'c': 3 } }] };

_.update(object, 'a[0].b.c', function(n) { return n * n; });
console.log(object.a[0].b.c);
// => 9

_.update(object, 'x[0].y.z', function(n) { return n ? n + 1 : 0; });
console.log(object.x[0].y.z);
// => 0
```

## set

<Heading level="2">
    set
</Heading>

<Paragraph>
    <Mark bold>_.set(object, path, value)</Mark>
</Paragraph>

<p>设置 object 中 path 路径对应的属性值。如果 path 的某一段不存在，会自动创建。缺失的索引属性会创建为数组，其他缺失属性会创建为普通对象。可通过 <Link href="#setwith">_.setWith</Link> 自定义路径创建。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：该方法会修改 object。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：要修改的对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            path
        </TableCell>
        <TableCell>
            (Array|string)：要设置的属性路径
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            value
        </TableCell>
        <TableCell>
            (*)：要设置的值
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    返回 object。
</Paragraph>

### 示例
```js
var object = { 'a': [{ 'b': { 'c': 3 } }] };

_.set(object, 'a[0].b.c', 4);
console.log(object.a[0].b.c);
// => 4

_.set(object, ['x', '0', 'y', 'z'], 5);
console.log(object.x[0].y.z);
// => 5
```

## unset

<Heading level="2">
    unset
</Heading>

<Paragraph>
    <Mark bold>_.unset(object, path)</Mark>
</Paragraph>

<p>移除 object 中 path 路径对应的属性。</p>

<Callout icon="ℹ️" blockColor="light_blue" borderColor="blue">
    注意：该方法会修改 object。
</Callout>

### 参数
<Table>
    <TableRow>
        <TableCell>
            object
        </TableCell>
        <TableCell>
            (Object)：要修改的对象
        </TableCell>
    </TableRow>
    <TableRow>
        <TableCell>
            path
        </TableCell>
        <TableCell>
            (Array|string)：要移除的属性路径
        </TableCell>
    </TableRow>
</Table>

### 返回值
<Paragraph>
    如果属性被成功删除返回 true，否则返回 false。
</Paragraph>

### 示例
```js
var object = { 'a': [{ 'b': { 'c': 7 } }] };
_.unset(object, 'a[0].b.c');
// => true

console.log(object);
// => { 'a': [{ 'b': {} }] };

_.unset(object, ['a', '0', 'b', 'c']);
// => true

console.log(object);
// => { 'a': [{ 'b': {} }] };
```
