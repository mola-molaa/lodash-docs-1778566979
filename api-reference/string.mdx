# String Methods

Lodash 提供了一系列用于处理字符串的实用方法，涵盖格式转换、内容检查、填充截断、替换拆分等场景。本文档详细说明各字符串方法的用法、参数、返回值及示例。

## camelCase

将字符串转换为驼峰命名法（camelCase）。

### 语法
```
_.camelCase([string=''])
```

### 参数
1. `[string='']` *(string)*: 要转换的字符串。

### 返回值
*(string)*: 返回驼峰格式的字符串。

### 示例
```js
_.camelCase('Foo Bar');
// => 'fooBar'

_.camelCase('--foo-bar--');
// => 'fooBar'

_.camelCase('__FOO_BAR__');
// => 'fooBar'
```

## endsWith

检查字符串是否以指定的目标字符串结尾。

### 语法
```
_.endsWith([string=''], [target], [position=string.length])
```

### 参数
1. `[string='']` *(string)*: 要检查的字符串。
2. `[target]` *(string)*: 要搜索的目标字符串。
3. `[position=string.length]` *(number)*: 检查到此位置的字符为止。

### 返回值
*(boolean)*: 如果字符串以目标字符串结尾则返回 `true`，否则返回 `false`。

### 示例
```js
_.endsWith('abc', 'c');
// => true

_.endsWith('abc', 'b');
// => false

_.endsWith('abc', 'b', 2);
// => true
```

## pad

如果字符串长度小于指定长度，则在左右两侧填充字符。

### 语法
```
_.pad([string=''], [length=0], [chars=' '])
```

### 参数
1. `[string='']` *(string)*: 要填充的字符串。
2. `[length=0]` *(number)*: 填充后的总长度。
3. `[chars=' ']` *(string)*: 用于填充的字符。

### 返回值
*(string)*: 返回填充后的字符串。

### 示例
```js
_.pad('abc', 8);
// => '  abc   '

_.pad('abc', 8, '_-');
// => '_-abc_-_'

_.pad('abc', 3);
// => 'abc'
```

## template

创建一个编译后的模板函数，支持插值、HTML 转义、执行 JavaScript 代码等功能。

<Callout icon="⚠️" blockColor="yellow" borderColor="light_orange">
    <Mark bold>安全提示</Mark>：`_.template` 存在安全风险，不建议使用，Lodash v5 版本将会移除该方法，请避免处理不可信的输入。
</Callout>

### 语法
```
_.template([string=''], [options={}])
```

### 参数
1. `[string='']` *(string)*: 模板字符串。
2. `[options={}]` *(Object)*: 配置对象。
3. `[options.escape=_.templateSettings.escape]` *(RegExp)*: HTML 转义分隔符。
4. `[options.evaluate=_.templateSettings.evaluate]` *(RegExp)*: 执行代码分隔符。
5. `[options.imports=_.templateSettings.imports]` *(Object)*: 导入到模板中的自由变量对象。
6. `[options.interpolate=_.templateSettings.interpolate]` *(RegExp)*: 插值分隔符。
7. `[options.sourceURL='lodash.templateSources[n]']` *(string)*: 编译模板的 sourceURL。
8. `[options.variable='obj']` *(string)*: 数据对象的变量名。

### 返回值
*(Function)*: 返回编译后的模板函数。

### 示例
```js
// 使用插值分隔符创建编译模板
var compiled = _.template('hello <%= user %>!');
compiled({ 'user': 'fred' });
// => 'hello fred!'

// 使用 HTML 转义分隔符转义数据属性值
var compiled = _.template('<b><%- value %></b>');
compiled({ 'value': '<script>' });
// => '<b>&lt;script&gt;</b>'

// 使用执行分隔符运行 JavaScript 并生成 HTML
var compiled = _.template('<% _.forEach(users, function(user) { %><li><%- user %></li><% }); %>');
compiled({ 'users': ['fred', 'barney'] });
// => '<li>fred</li><li>barney</li>'
```

## truncate

如果字符串长度超过指定最大长度，则截断字符串，末尾替换为省略字符串（默认为 `...`）。

### 语法
```
_.truncate([string=''], [options={}])
```

### 参数
1. `[string='']` *(string)*: 要截断的字符串。
2. `[options={}]` *(Object)*: 配置对象。
3. `[options.length=30]` *(number)*: 最大字符串长度。
4. `[options.omission='...']` *(string)*: 表示文本被省略的字符串。
5. `[options.separator]` *(RegExp|string)*: 截断到的分隔符模式。

### 返回值
*(string)*: 返回截断后的字符串。

### 示例
```js
_.truncate('hi-diddly-ho there, neighborino');
// => 'hi-diddly-ho there, neighbo...'

_.truncate('hi-diddly-ho there, neighborino', {
  'length': 24,
  'separator': ' '
});
// => 'hi-diddly-ho there,...'

_.truncate('hi-diddly-ho there, neighborino', {
  'length': 24,
  'separator': /,? +/
});
// => 'hi-diddly-ho there...'

_.truncate('hi-diddly-ho there, neighborino', {
  'omission': ' [...]'
});
// => 'hi-diddly-ho there, neig [...]'
```
