<Heading level="1">
    模块格式
</Heading>

Lodash 支持多种模块格式，你可以根据项目技术栈选择最合适的引入方式。

<BulletedList>
    标准 CommonJS / UMD 包：<Mark color="blue">lodash</Mark>，可直接用于 Node.js 或浏览器 <Mark bold>script</Mark> 标签引入。
</BulletedList>
<BulletedList>
    ES Module 包：<Mark color="blue">lodash-es</Mark>，适用于现代构建工具与 Tree Shaking。
</BulletedList>
<BulletedList>
    函数式编程包：<Mark color="blue">lodash/fp</Mark>，不可变、自动柯里化方法集合。
</BulletedList>
<BulletedList>
    AMD 版本：<Mark color="blue">lodash-amd</Mark>，适用于 RequireJS 等 AMD 环境。
</BulletedList>
<BulletedList>
    按方法拆分包：npm 上提供 <Mark color="blue">lodash-modularized</Mark> 系列，可按需安装单个方法包。
</BulletedList>

<Heading level="1">
    安装与引入
</Heading>

<Heading level="2">
    浏览器直接引入
</Heading>

通过 <Mark bold>script</Mark> 标签加载 UMD 构建产物后，全局会暴露 <Mark color="blue">_</Mark> 变量：

```html
<script src="lodash.js"></script>
```

<Heading level="2">
    npm 安装
</Heading>

使用 npm 安装完整版 Lodash：

```shell
$ npm i --save lodash
```

安装完成后即可在 Node.js 或构建工具中引入：

```js
// 完整构建
var _ = require('lodash');

// 核心构建
var _ = require('lodash/core');

// FP 构建（不可变、自动柯里化）
var fp = require('lodash/fp');

// 按分类引入
var array = require('lodash/array');
var object = require('lodash/fp/object');
```

<Heading level="2">
    ES Module 引入
</Heading>

如果你的项目使用 ES Module，推荐安装 <Mark color="blue">lodash-es</Mark> 以获得更好的 Tree Shaking 支持：

```shell
$ npm i --save lodash-es
```

```js
import _ from 'lodash-es';
// 或按需引入
import { debounce, throttle } from 'lodash-es';
import { curryN } from 'lodash-es/fp';
```

<Heading level="2">
    构建工具中的按需引入
</Heading>

为减少最终 bundle 体积，建议在打包工具中按方法路径引入，实现精确加载。

<Heading level="3">
    Browserify / Rollup
</Heading>

这类工具支持直接解析 Lodash 内部路径，可按方法引入：

```js
var at = require('lodash/at');
var curryN = require('lodash/fp/curryN');
```

配合 <Link href="https://www.npmjs.com/package/babel-plugin-lodash">babel-plugin-lodash</Link> 还能进一步简化写法。

<Heading level="3">
    Webpack
</Heading>

Webpack 环境下同样支持路径引入，也可以结合以下工具优化：

<BulletedList>
    使用 <Mark color="blue">lodash-webpack-plugin</Mark> 进一步剔除未使用功能。
</BulletedList>
<BulletedList>
    搭配 <Mark color="blue">babel-plugin-lodash</Mark> 实现自动按需引入。
</BulletedList>

```js
// 按需引入示例
var at = require('lodash/at');
var curryN = require('lodash/fp/curryN');
```
